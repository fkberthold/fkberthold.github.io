Spanish Speakers: Pomodoro means Tomato

Way of tracking tasks over time.

Digression: What Is Pomodoro?

Planning

  • Assemble a list of tasks
  • Estimate the number of pomos per task
  • Select your planned tasks for the day

Start each day with a plan. Sharper, more organized. Know the first I'm going to do when I walk in. That leads to the rest of the day.

What's a Pomo?

Roughly 25 minutes

  • Thursday Chores (1) ✓
  • Put together a presentation on pomodoros (3) ✓ ✓
  • Write Automation for Facebook Ads (7)

Note the rough process:

  • Pick a task off the list
  • Start the timer
  • Work on it, and only it for that time
  • Once you're done, put a mark next to the list

FEATURE: Can be done with pencil, paper and a kitchen timer.

Why I Use These Tools

  • Estimates are easy to make and track
  • Lots of ways to win
  • Clear Boundaries
  1. It's really obvious when I've messed up an estimate and I can use past results to improve my estimates.
  2. Both time and task goals give something to shoot for, and smnall wins throughout the day.
  3. It gives me clear boundaries for when I'm working on just one thing, and a time when I'll be able to step away and do other things.
